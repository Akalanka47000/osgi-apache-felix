# osgi-equinox
A project built on top of Apache Equinox for an assignment

## Useful commands

```bash
install file:K:/Repositories/osgi-equinox/dictionary-service/bin/dictionary/service/dictionary-service.jar
install file:K:/Repositories/osgi-equinox/dictionary-client/bin/dictionary/client/dictionary-client.jar

uninstall file:K:/Repositories/osgi-equinox/dictionary-service/bin/dictionary/service/dictionary-service.jar
uninstall file:K:/Repositories/osgi-equinox/dictionary-client/bin/dictionary/client/dictionary-client.jar

start file:K:/Repositories/osgi-equinox/dictionary-service/bin/dictionary/service/dictionary-service.jar
start file:K:/Repositories/osgi-equinox/dictionary-client/bin/dictionary/client/dictionary-client.jar   
```
